HTML Exemplo Flexbox

Especialização CSS: Flexbox (Unidimensional)

O Flexbox é ideal para alinhamento em **uma única dimensão** (linha ou coluna) e é perfeito para centrar elementos, sem a necessidade de hacks como o `transform` ou `float`.

ESTE ITEM ESTÁ CENTRALIZADO

Código Aplicado:

.flex-container {
 display: flex;
 justify-content: center; /* HORIZONTAL */
 align-items: center; /* VERTICAL */
}
