Especialização CSS: Grid Layout (Bidimensional)

O CSS Grid é ideal para layouts de página, pois permite controlar a disposição dos itens em **linhas e colunas simultaneamente**, de forma limpa e declarativa.

ÁREA DE CABEÇALHO (Ocupa 3 colunas)
Coluna A (1/4 da largura)
Coluna B (2/4 da largura - Conteúdo Principal)
Coluna C (1/4 da largura - Sidebar)

Código Aplicado:

.grid-container {
 display: grid;
 /* Define 3 colunas, com a do meio sendo o dobro */
 grid-template-columns: 1fr 2fr 1fr;
}
.item-topo {
 /* Faz o item iniciar na coluna 1 e se estender por 3 colunas */
 grid-column: 1 / span 3;
}
